Protected Pages:
---------------
dashboard.html         - Main dashboard view
portfolio.html         - Investment portfolio
goals.html             - Financial goals tracker
settings.html          - Account settings
wealthbot.html         - AI financial assistant

Storage Keys:
------------
wealthos_user          - Array of registered users
wealthos_session       - Current user session

Session Duration:
----------------
Default:               24 hours
With "Remember me":    30 days
Auto-expires:          Yes

Test Flow:
---------
1. Open /Prototype/index.html
2. Click "Get Started"
3. Fill signup form
4. Complete onboarding (or skip)
5. Access dashboard
6. Navigate to other protected pages

Browser DevTools:
----------------
View Data:
  - Press F12
  - Application → Local Storage
  - Check wealthos_user & wealthos_session

Clear Session:
  - Delete wealthos_session key
  - Refresh page → Redirects to login
